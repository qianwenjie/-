# 学生端手机端考试系统

> 基于 Vue 3 + Vite + Vant 4 的移动端考试应用

## 项目概述

这是考试系统的学生端手机应用，主要功能包括：
- 考试列表与详情查看
- 人脸识别验证
- 在线答题（支持7种题型）
- 答题卡功能
- 提交与成绩查看

## 技术栈

- **Vue 3.5+** - 渐进式 JavaScript 框架
- **Vite 7.2+** - 下一代前端构建工具
- **Vant 4.9+** - 移动端 UI 组件库
- **Pinia 3.0+** - Vue 状态管理库
- **Vue Router 4.6+** - 官方路由管理器
- **Axios 1.13+** - HTTP 客户端

## 状态管理

### Exam Store

管理考试列表和当前考试信息

### Answer Store

管理答题状态，包括：
- 答案数据
- 标记题目
- 当前题目索引
- 倒计时
- 自动保存

### User Store

管理用户信息和登录状态

## API 接口

当前使用 Mock 数据进行开发，生产环境需要对接真实 API。

Mock 模式开关：`src/api/exam.js` 中的 `USE_MOCK` 变量

## 注意事项

1. **离线支持**：答题数据会自动保存到 LocalStorage
2. **自动保存**：每30秒自动同步答案到服务器
3. **网络恢复**：网络恢复后自动同步未保存的数据
4. **防止数据丢失**：页面关闭前会提示保存
